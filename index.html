About Confox

Confox is a collection of open-source projects on GitHub, focusing on modern web development tools, themes, and resources. These repositories provide developers with flexible, customizable solutions for enhancing their projects.

Repositories

ByeBye Script

ByeBye is a Python script designed to securely delete and overwrite data on drives in a system, providing a simple interface for users to select a drive and make its contents unrecoverable.

Languages: Python

Project NEXIS

Project NEXIS is a personal AI virtual assistant powered by advanced neural networks, designed to assist users with scheduling, reminders, and more.

Languages: HTML, CSS, JavaScript
